dotsplat

uses git in concert with symlinks to track your precious dotfiles

  • Installation -- Create an empty git repo named dotfiles

    $curl -fsSL http://install.hovr2pi.org/dotsplat | bash 
    $source $HOME/.dotsplat.git/dotsplat.sh 
    $dotsplat clone https://github.com/$USERNAME/dotfiles

  • Add files to be tracked

    $ dotsplat track $HOME/.somedotfile1 # adding a file to dotsplat
			$ dotsplat track $HOME/.somedotfile2 # tracking a second file
			$ dotsplat cd dotfiles # pops you into the git repo backend for your dotfiles
			$ git commit -a 
			$ git push -u origin master
